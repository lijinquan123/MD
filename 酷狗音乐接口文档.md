每个接口基本都会要校验参数signature,使用MD5签名算法

```python
def calculate_signature(params: dict, data: str = '', key: str = None):
    """
    计算signature
    :param params: 网络请求的params
    :param data: 网络请求的data,必须是字符串格式
    :param key: 加密key,一般是OIlwieks28dk2k092lksi2UIkp
    :return:
    """
    if 'srcappid' in params:
        params.pop('appid')
    if key is None:
        key = 'OIlwieks28dk2k092lksi2UIkp'
    sorted_params = [f'{k}={v}' for k, v in sorted(params.items(), key=lambda x: x[0])]
    sorted_params.insert(0, key)
    sorted_params.append(data)
    sorted_params.append(key)
    return md5(''.join(sorted_params))
```
